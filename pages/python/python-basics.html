Python Basics for Hackers

Introduction to Python for Security

Python is one of the most popular programming languages for cybersecurity professionals. Its simplicity, extensive libraries, and powerful features make it ideal for security tasks.

Core Theory

Python's philosophy "readability counts" makes it perfect for security work where code needs to be quickly understood, modified, and shared among team members. Think of Python as the "Swiss Army knife" of programming languages.

Python's Role in Cybersecurity

Python has become the de facto standard in cybersecurity for several reasons:

Technical Advantages

  • Interpreted Language: No compilation needed, faster development
  • Dynamic Typing: Flexible variable types, rapid prototyping
  • Rich Standard Library: Built-in modules for networking, crypto, etc.
  • Cross-Platform: Works on Windows, Linux, macOS
  • Memory Management: Automatic garbage collection
  • Rapid Development: Quick prototyping
  • Network Programming: Built-in socket support
  • Automation: Perfect for repetitive tasks
  • Integration: Works with other tools
  • Community: Large security community

3. Cryptography

from Crypto.Cipher import AES from Crypto.Random import get_random_bytes import hashlib # Hash functions def hash_password(password): return hashlib.sha256(password.encode()).hexdigest() # AES Encryption def encrypt_data(data, key): cipher = AES.new(key, AES.MODE_EAX) ciphertext, tag = cipher.encrypt_and_digest(data.encode()) return cipher.nonce, ciphertext, tag def decrypt_data(nonce, ciphertext, tag, key): cipher = AES.new(key, AES.MODE_EAX, nonce=nonce) plaintext = cipher.decrypt_and_verify(ciphertext, tag) return plaintext.decode() # Example usage key = get_random_bytes(16) nonce, ciphertext, tag = encrypt_data("Secret message", key) decrypted = decrypt_data(nonce, ciphertext, tag, key) print(f"Decrypted: {decrypted}")

Practical Security Scripts

1. Simple Port Scanner

#!/usr/bin/env python3 import socket import sys from datetime import datetime def scan_port(target, port): try: sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM) sock.settimeout(1) result = sock.connect_ex((target, port)) sock.close() return result == 0 except socket.gaierror: return False def main(): if len(sys.argv) != 2: print("Usage: python3 scanner.py ") sys.exit(1) target = sys.argv[1] print(f"Starting port scan on {target}") print(f"Time started: {datetime.now()}") print("-" * 50) try: for port in range(1, 1001): if scan_port(target, port): print(f"Port {port}: Open") except KeyboardInterrupt: print("\nScan interrupted by user") sys.exit(1) print(f"Scan completed at {datetime.now()}") if __name__ == "__main__": main()

#!/usr/bin/env python3 import random import string import secrets def generate_password(length=12, include_symbols=True): """Generate a secure random password""" characters = string.ascii_letters + string.digits if include_symbols: characters += "!@#$%^&*" password = ''.join(secrets.choice(characters) for _ in range(length)) return password def generate_wordlist(base_word, variations=100): """Generate password variations for testing""" wordlist = [] # Common variations for i in range(variations): variation = base_word # Add numbers if random.choice([True, False]): variation += str(random.randint(0, 999)) # Capitalize if random.choice([True, False]): variation = variation.capitalize() # Add symbols if random.choice([True, False]): variation += random.choice("!@#$%") wordlist.append(variation) return list(set(wordlist)) # Remove duplicates # Examples print("Secure passwords:") for _ in range(5): print(generate_password(16)) print("\nPassword variations for 'admin':") variations = generate_wordlist("admin", 10) for var in variations[:10]: print(var)
