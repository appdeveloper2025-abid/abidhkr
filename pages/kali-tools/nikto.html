Nikto Web Vulnerability Scanner

What is Nikto?

Nikto is an open-source web server scanner that performs comprehensive tests against web servers for multiple items, including dangerous files, outdated server software, and server configuration issues.

Learning Objective: Master web server vulnerability scanning, configuration assessment, and security testing using Nikto's comprehensive testing capabilities.

Key Features

  • 6700+ Tests: Comprehensive vulnerability database
  • Server Detection: Web server fingerprinting
  • Plugin System: Modular testing approach
  • SSL Support: HTTPS scanning capabilities
  • Multiple Formats: Various output options
  • Proxy Support: Works with web proxies
  • Authentication: Basic and NTLM auth
  • Cookie Support: Session-based scanning

What Nikto Detects

Server Issues
  • Outdated software versions
  • Default files and directories
  • Insecure configurations
  • Missing security headers
Dangerous Files
  • Backup files
  • Test files
  • Admin interfaces
  • Debug pages
Vulnerabilities
  • Known CVEs
  • XSS vulnerabilities
  • SQL injection points
  • Directory traversal

Installation & Setup

Kali Linux

# Nikto is pre-installed in Kali Linux nikto -Version # Update Nikto database nikto -update # Manual installation sudo apt update sudo apt install nikto

Ubuntu/Debian

# Install from repositories sudo apt update sudo apt install nikto # Install dependencies sudo apt install perl libnet-ssleay-perl openssl libauthen-pam-perl libpam-runtime libio-pty-perl apt-show-versions

From Source

# Clone from GitHub git clone https://github.com/sullo/nikto.git cd nikto/program # Run directly ./nikto.pl -h target.com # Make executable globally sudo ln -s /path/to/nikto/program/nikto.pl /usr/local/bin/nikto

Basic Scanning Operations

Simple Scans

# Basic scan nikto -h target.com # Scan specific port nikto -h target.com -p 8080 # Scan multiple ports nikto -h target.com -p 80,443,8080 # HTTPS scan nikto -h https://target.com # Scan with verbose output nikto -h target.com -v

Target Specification

# Single host nikto -h 192.168.1.100 # Multiple hosts nikto -h 192.168.1.100,192.168.1.101 # Host range nikto -h 192.168.1.100-110 # From file nikto -h hosts.txt # Subnet scan nmap -sL 192.168.1.0/24 | grep "Nmap scan report" | awk '{print $5}' > hosts.txt nikto -h hosts.txt

Authentication

# Basic authentication nikto -h target.com -id username:password # NTLM authentication nikto -h target.com -id domain\\username:password # Cookie-based authentication nikto -h target.com -C "PHPSESSID=abc123; user=admin" # Custom headers nikto -h target.com -H "Authorization: Bearer token123"

Advanced Scanning Options

Scan Tuning

# Scan specific categories nikto -h target.com -T 1 # Interesting files nikto -h target.com -T 2 # Misconfiguration nikto -h target.com -T 3 # Information disclosure nikto -h target.com -T 4 # Injection (XSS/Script/HTML) nikto -h target.com -T 5 # Remote file retrieval nikto -h target.com -T 6 # Denial of service nikto -h target.com -T 7 # Remote file retrieval (server wide) nikto -h target.com -T 8 # Command execution (remote shell) nikto -h target.com -T 9 # SQL injection nikto -h target.com -T 0 # File upload nikto -h target.com -T a # Authentication bypass nikto -h target.com -T b # Software identification nikto -h target.com -T c # Remote source inclusion # Multiple categories nikto -h target.com -T 1,2,3

Performance Options

# Timeout settings nikto -h target.com -timeout 10 # Maximum scan time nikto -h target.com -maxtime 3600 # Delay between requests nikto -h target.com -Pause 2 # User agent nikto -h target.com -useragent "Mozilla/5.0 (compatible; Nikto)" # Custom user agent nikto -h target.com -useragent "Custom Scanner 1.0"

Proxy and SSL

# Use proxy nikto -h target.com -useproxy http://proxy:8080 # Ignore SSL certificate errors nikto -h https://target.com -ssl # Force SSL version nikto -h target.com -ssl -port 443 # Disable SSL checks nikto -h https://target.com -nossl

Output Formats & Reporting

Output Options

# Save to file (text format) nikto -h target.com -o results.txt # HTML report nikto -h target.com -o results.html -Format html # XML output nikto -h target.com -o results.xml -Format xml # CSV format nikto -h target.com -o results.csv -Format csv # Multiple formats nikto -h target.com -o results -Format html,xml,txt

Report Customization

# No banner in output nikto -h target.com -nobanner # Display only vulnerabilities nikto -h target.com -Display V # Display options: # 1 - Show redirects # 2 - Show cookies received # 3 - Show all 200/OK responses # 4 - Show URLs which require authentication # V - Verbose output # Custom display nikto -h target.com -Display 1,2,V

Database Updates

# Update plugin database nikto -update # List all plugins nikto -list-plugins # Show database info nikto -dbcheck # Force database update nikto -update -D V

Plugins & Advanced Features

Plugin Management

# List available plugins nikto -list-plugins # Use specific plugins nikto -h target.com -Plugins "apacheusers,cgi" # Exclude plugins nikto -h target.com -Plugins "@@DEFAULT" -PluginTimeout 60 # Plugin categories: # - apacheusers: Enumerate Apache users # - cgi: CGI scanning # - headers: HTTP header analysis # - httpoptions: HTTP methods testing # - msgs: Custom message testing # - outdated: Outdated software detection # - paths: Path discovery # - robots: robots.txt analysis

Custom Scans

# Scan specific directory nikto -h target.com -root /admin/ # Custom port range nikto -h target.com -p 1-1000 # Evasion techniques nikto -h target.com -evasion 1 # Evasion modes: # 1 - Random URI encoding # 2 - Directory self-reference # 3 - Premature URL ending # 4 - Prepend long random string # 5 - Fake parameter # 6 - TAB as request spacer # 7 - Change case # 8 - Use Windows directory separator # Multiple evasion techniques nikto -h target.com -evasion 1,2,3

Integration Examples

# Combine with Nmap nmap -p 80,443 --open target.com | grep "open" | awk '{print $1}' | nikto -h - # Scan from Nmap XML nmap -p 80,443 -oX scan.xml target.com nikto -h scan.xml # Integration with Burp Suite # 1. Configure Nikto to use Burp as proxy nikto -h target.com -useproxy http://127.0.0.1:8080 # Automated scanning script #!/bin/bash for host in $(cat targets.txt); do echo "Scanning $host" nikto -h $host -o "nikto_$host.html" -Format html done

Common Use Cases

# Quick vulnerability assessment nikto -h target.com -T 1,2,3,4,9 -o quick_scan.html -Format html # Comprehensive security audit nikto -h target.com -T 1,2,3,4,5,6,7,8,9,0,a,b,c -o full_audit.xml -Format xml # Stealth scanning nikto -h target.com -T 1,2,3 -timeout 20 -Pause 5 -evasion 1,2,7 # SSL/TLS specific testing nikto -h https://target.com -ssl -T 2,3 -Plugins "headers,httpoptions"
